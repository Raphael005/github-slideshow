# Getting Started

Welcome to GitHub Slideshow! This guide will help you get up and running.

## Prerequisites

- Git installed on your machine
- A GitHub account

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Raphael005/github-slideshow.git

# Navigate to the project
cd github-slideshow

# Create a new branch
git checkout -b my-feature
```
